# Runbooks

Short operational guides for Dashboardy platform delivery.

| Runbook | Description |
|---------|-------------|
| [deploy-staging.md](deploy-staging.md) | Cut a release tag and watch staging deploy; recover from failures. |
| [promote-production.md](promote-production.md) | Manual approval gate and production rollout using the same images as staging. |
| [rollback.md](rollback.md) | Return production to a previous known-good tag; forward-only migrations. |
| [github-environments-setup.md](github-environments-setup.md) | One-time GitHub Environments and secrets for staging and production workflows. |
